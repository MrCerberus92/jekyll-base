Domain management

Domains

Set a custom domain and manage domain aliases.

Custom domain

No custom domain set

By default, your site is accessible at cerberus4multimedia.netlify.com.

You can change the Netlify subdomain by changing the site’s name in General settings.

If you own a custom domain, you can add it to your site by clicking Add a custom domain.

Learn more about custom domains in the documentation

Domain aliases

No custom domain set

Set a primary custom domain to enable domain alias functionality, allowing you to point more custom domains to this site.

HTTPS

Enable automatic TLS certificates with Let’s Encrypt, or use your own certificate.

SSL/TLS certificate

A custom domain is required to provision a certificate

You’ll need to set a custom domain before your can enable HTTPS.

Custom domain panel ↑

Force HTTPS

You must have a certificate installed to force HTTPS

Ensure you first have a custom domain set, and then ensure you selected to provision a free Let’s Encrypt certificate or have provided your own.

DNS

Manage DNS records, or turn deployed branches into subdomains.

DNS records

DNS panel requires Netlify DNS management

The Netlify DNS panel is available on sites which opt to use the Netlify DNS with their provided custom domain. Ensure you have a custom domain set and have chosen to use Netlify DNS.

Branch subdomains

Branch subdomains require DNS management
